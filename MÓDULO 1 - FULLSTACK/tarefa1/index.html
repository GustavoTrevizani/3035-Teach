Introdução ao Processo Seletivo

Esta tarefa faz parte do primeiro módulo do curso para estágio na 3035 TECH. O foco é em conceitos básicos de HTML, como tags estruturais e atributos para criar páginas web funcionais.

  1. Objetivo: Demonstrar habilidades em HTML puro.
  2. Conteúdo: Seções de texto, navegação e mídia.
  3. Próximos passos: Módulos de CSS e JavaScript.

Detalhes Técnicos do HTML

O HTML usa <tags> para estruturar conteúdo. Atributos como id e class permitem navegação e estilização futura. Caracteres reservados, como & (e comercial), garantem formatação correta.

  • Tags Estruturais: <header>, <main>, <footer>.
  • Elementos de Texto: Headings (<h1> a <h6>), parágrafos e listas.
  • Mídia: Vídeos e imagens com controles e acessibilidade.
